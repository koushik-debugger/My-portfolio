# My Portfolio

Welcome to my portfolio repository! This is a showcase of my skills and projects as a developer.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Easy to Customize**: Simple structure to add your own projects and information

## File Structure

```
├── index.html      # Main HTML file
├── styles.css      # CSS styling
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Customize the content with your own information:
   - Update the header and navigation
   - Modify the about section
   - Add your projects to the projects section
   - Update contact information

## Customization

### Update Personal Information
Edit `index.html` to add your name, bio, and project details.

### Change Colors
Modify the color scheme in `styles.css`. The main colors are defined at the top of the file.

### Add Projects
Add new project cards in the projects section by duplicating the project-card div.
